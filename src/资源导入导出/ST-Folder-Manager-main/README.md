# 📁 酒馆资源管理器 (ST-Folder-Manager)

SillyTavern 第三方扩展插件，提供类似浏览器收藏夹风格的双栏资源管理界面，支持对**角色卡**、**预设**、**世界书**、**正则脚本**、**美化（主题）**、**背景**、**User（用户设定/Persona）**和**快速回复集（QR）**进行文件夹分组管理。

> 作者：QwQ&AI ｜ 版本：1.0.0

---

## ✨ 核心功能

### 🗂️ 八大资源统一管理

通过顶部标签页在八种资源类型之间切换：

- **角色卡** — 基于酒馆标签系统，将角色卡归入文件夹
- **预设** — 对当前 API 的预设进行分组管理，点击即可应用，支持**预设条目激活分组**，并可与世界书进行**条目互通缝合**
- **世界书** — 对世界书进行分组管理，点击可打开编辑器，并可与预设进行**条目互通缝合**
- **正则脚本** — 对正则脚本进行文件夹分组管理，支持**全局 / 角色 / 预设**三端查看与互通、启用/禁用切换、激活分组
- **美化** — 对 SillyTavern 主题进行分组管理，点击即可应用主题，支持绑定背景
- **背景** — 对聊天背景图片进行分组管理，点击即可应用背景，支持设置默认背景
- **User** — 对用户设定（Persona）进行分组管理，点击即可切换 Persona，显示绑定角色/群组标签。支持点击小三角展开查看每个 User 的详情栏，显示标题、绑定、具体设定与备注；标题和具体设定可直接点击编辑按钮修改，绑定区提供“默认 / 角色 / 聊天”三个原生风格按钮进行快速绑定操作。选中绑定了多个 User 的角色时，弹窗经过优化，更容易分清各个 User 的对应关系
- **快速回复集（QR）** — 对快速回复集进行文件夹分组管理，支持展开查看每个集包含的快速回复项，并支持激活分组绑定角色卡/预设

### 📂 无限层级文件夹树

- 支持无限嵌套的树形文件夹结构
- 左侧树形导航 + 右侧内容列表的双栏布局
- 展开/收起全部、面包屑路径显示
- 每个文件夹显示递归计数

### 🏷️ 标签自动同步（角色卡）

- **首次加载**自动导入酒馆中所有已有标签为文件夹
- **每次打开**自动检测新标签并导入，新标签带"新"徽章高亮
- 一键导入所有未注册标签
- 子文件夹自动使用路径前缀命名，避免标签重名冲突

### ☑️ 多选操作

- **全选**：一键选中当前视图中的所有资源项
- **框选**：点击起点再点击终点（或 Shift+点击），范围内资源全部选中
- 仅作用于资源（角色卡/预设/世界书/正则/美化/背景/User/QR），不影响文件夹
- 搜索功能启用时（搜索栏有内容），多选同样可用
- **PC 端多选拖拽**：选中多个资源后，拖拽其中任意一个即可带动所有选中项一起移动，拖拽时显示"📦 共 X 项"浮动提示
- **双击文件夹移入**：多选模式下，双击左侧树或右侧列表中的任意文件夹，即可将所有选中资源批量移入该文件夹（适用于全部七种资源类型）

### 🖱️ 拖拽操作

- **PC 端**：原生 HTML5 拖拽，支持文件夹和角色卡/预设/世界书
- **移动端**：长按触发触摸拖拽，带幽灵预览元素；多选拖拽时显示"共 X 项"
- **三区域拖放**：拖到节点上方 25% = 排在前面，中间 50% = 嵌套进入，下方 25% = 排在后面
- 自动检测循环嵌套并阻止
- 拖到"未归类"节点可将项目移出所有文件夹

### 🔀 移动 / 复制模式

- 顶部切换按钮，在移动模式和复制模式之间切换
- **移动模式**：拖拽角色会从原文件夹移除，只保留在目标文件夹
- **复制模式**：拖拽角色会保留原位置，同时添加到目标文件夹
- 角色卡和预设/世界书各自独立的模式状态

### 📝 资源备注

- 在**美化**、**背景**、**预设**、**世界书**、**User**、**QR**标签页可为每个资源项添加**备注**信息，方便记录用途或特点
- 支持进入**备注编辑模式**，批量为多个资源项添加或修改备注
- 每行资源右侧有铅笔按钮，可单独快速编辑备注
- 备注内容显示在资源名旁边，搜索时也会匹配备注内容
- 备注数据随导入/导出一起备份和恢复

### 🔗 美化主题绑定背景

- 在美化标签页中，每个主题行右侧有**锁链图标**按钮，可将当前使用的背景绑定到该主题
- 切换美化主题时，自动应用绑定的背景图片，无需手动切换
- 已绑定的主题锁链图标显示为**绿色高亮**，hover 可查看绑定的背景名称
- 再次点击锁链图标可**更新绑定**或**解除绑定**
- 绑定数据随导入/导出一起备份和恢复

### 🖼️ 默认背景

- 在背景标签页工具栏中有**图片图标**按钮，可将当前使用的背景设为默认背景
- 切换到**没有绑定背景**的美化主题时，自动应用默认背景
- 已设置默认背景时按钮显示为**绿色高亮**
- 支持设置、清除默认背景，数据随导入/导出一起备份和恢复
- 背景删除或重命名时自动同步默认背景设置

### ✏️ 快速重命名（预设 & 世界书 & 美化 & 背景 & QR）

- 在**预设**、**世界书**、**美化**、**背景**和**QR**标签页，每行资源名旁边有重命名图标按钮（光标图标，hover 黄色高亮），点击即可快速重命名单个资源
- 支持进入**重命名模式**，批量选中多个资源后统一操作：
  - **添加前缀 / 后缀**：为所有选中项批量添加指定文本
  - **删除前缀 / 后缀**：批量移除选中项的公共前缀或后缀
  - **自动检测**：弹窗中自动识别选中项的公共前缀和后缀，点击即可快速填入
- 美化主题重命名通过保存新名称+删除旧名称实现；背景重命名使用原生 API，自动保留文件扩展名
- 重命名后自动同步文件夹分配、收藏状态和备注信息，无需手动调整

### ✏️ 角色卡快速编辑

- 在角色卡标签页可进入**编辑模式**，批量修改角色卡的**作者名**和**版本号**
- 支持多选（全选、框选、Shift+点击）后统一编辑
- 每行角色卡右侧有铅笔按钮，可单独快速编辑
- 蓝色主题色（#89b4fa）区分于导出（绿色）和删除（红色）模式

### 📖 角色卡具体设定展开

- 仅在**角色卡**标签页中，每个角色名左侧新增一个**小三角展开按钮**
- 点击后可在角色行下方展开查看该角色卡的**具体设定内容**，界面风格参考“查看聊天记录”的展开子面板
- 展开面板会显示角色卡中的常见字段，例如：**描述**、**性格**、**场景**、**首条消息**、**示例对话**，以及存在时的其他附加字段
- **描述**、**性格**、**场景**、**首条消息**、**示例对话**、**作者备注**、**系统提示词**、**历史后指令** 均可直接点击右侧编辑按钮进行快速修改
- 当角色卡存在**多个开场白**时，“首条消息”区域会显示左右切换按钮，可在多条开场白之间循环查看
- 角色卡名称旁的小三角与左侧功能位的 **查看聊天记录** / **查看正则** 小三角相互独立，避免多个相同位置箭头造成混淆
- 三种展开内容可分别独立展开/收起，互不覆盖

### � 角色卡信息展示

- 角色卡列表中，角色名旁边自动显示**版本号**和**创作者**信息（读取自角色卡高级定义）
- 版本号在前、创作者在后，用 " · " 分隔
- 仅在角色卡设置了相关信息时才显示，不占额外行高

### ⭐ 收藏功能

- 角色卡、预设、世界书、正则、美化、背景、User、QR 均支持星标收藏
- 左侧树顶部固定"收藏"入口，快速查看所有收藏项
- 收藏视图中角色卡会显示所在文件夹路径

### 🔍 全局搜索

- 每个资源标签页都有独立的搜索栏
- 支持搜索**项目名称**或**文件夹名称**
- 搜索范围可选：**当前文件夹**（递归）或**全部**
- 搜索结果可点击直接导航到对应文件夹
- **模糊搜索**：所有标签页（含 User）均支持空格分词多关键词模糊匹配
- **User 关联搜索**：在 User 标签页搜索时，输入绑定角色/群组的名称也能找到对应的 User

### 👤 User 详情展开与快速编辑

- 在 **User** 标签页中，每个 User 名称旁提供**小三角展开按钮**，点击即可展开详情栏
- 详情栏会显示 **标题**、**绑定**、**具体设定** 和 **备注**；没有内容时显示 **无**
- **标题** 与 **具体设定** 右侧带有编辑按钮，可直接在详情栏内快速修改
- **绑定** 区提供带中文标签的 **默认 / 角色 / 聊天** 三个按钮，悬停说明为中文
- 绑定按钮采用接近酒馆原生 Persona 面板的图标风格，并复用原生绑定逻辑
- 普通列表与搜索结果列表中的 User 项都支持相同的展开与编辑交互

### 📊 排序功能

- 左栏：顶级文件夹 A→Z / Z→A 排序
- 右栏：子文件夹排序 + 项目名称排序
- 支持恢复自定义排序（排序快照机制）
- 关闭弹窗时若排序已更改，会弹出确认对话框（保留/撤回）

## ⚙️ 配置管理

点击右上角齿轮按钮进入配置面板：

### 按钮位置

- **固定在顶栏**：在酒馆顶部导航栏显示文件夹图标
- **浮动按钮**：可自由拖拽定位的悬浮按钮（支持 PC 鼠标拖拽和移动端长按拖拽）
- **魔术棒菜单**：将入口添加到底部输入栏旁的魔术棒扩展菜单中，不占用上下文视野也不影响顶部栏，适合移动端使用

### 自定义顶栏图标

- 顶栏模式下可自定义按钮图标，支持手动输入图标 URL
- **美化主题自动适配**：自动检测当前美化主题中的图标替换规则，一键从下拉菜单选择主题图标
- **主题切换自动监听**：切换美化主题后自动更新图标，无需手动操作（MutationObserver + 轮询双策略确保可靠检测）
- 未手动指定图标时，默认自动使用美化主题中「用户设定管理」按钮的图标

### 自定义布局

- 在设置面板中可自由配置**标签页**和**子功能按钮**的显示与排序
- **标签页管理**：支持隐藏/显示和拖拽排序八个标签页（角色卡/世界书/预设/正则/美化/背景/User/QR），至少保留一个
- **子功能管理**：每个标签页的工具栏按钮（导入/导出/删除/编辑备注/重命名/设置默认背景等）均可独立开关和排序
- 关闭设置页后立即生效，无需重新打开插件
- 支持一键**恢复默认**布局

### 🎨 插件自定义外观

- 可为插件界面单独配置**背景颜色 / 不透明度 / 文字颜色 / 边框颜色 / 强调色 / 背景模糊**
- 支持单独配置展开详情区域的**背景色、文字色、条目名称颜色**
- 以**当前酒馆主题**为单位分别保存外观配置，切换主题后自动套用对应样式
- 内置**深色 / 浅色 / 半透明 / 高对比度**四种快捷预设
- 支持保存“**我的预设**”，并可搜索、重命名、删除和一键复用
- 弹窗内提供**实时预览**、**单项重置**与**恢复默认外观**操作

### 🌐 界面语言切换

- 设置面板中可在**简体中文**与**繁體中文**之间切换
- 切换后重新打开插件即可生效
- 插件自身的弹窗、提示与确认框会随语言设置自动转换
- 为保证新增界面内容也能同步转换，插件会对自身新增 DOM 文本自动执行简繁转换

### 📱 移动端顶部栏避让

- 可开启“**移动端避让顶部栏**”，让插件弹窗从酒馆顶部栏下方展开
- 关闭后恢复为全屏覆盖模式
- 仅影响移动端显示，不影响桌面端

### 创建文件夹

- **单个创建**：输入名称，空格分隔可批量创建同级文件夹
- **批量创建**：多行缩进格式，一次性创建完整的文件夹树结构
  - 每 2 个空格代表一层嵌套
  - 支持智能缩进（Enter 自动缩进、Backspace 回退层级）
  - 创建前可预览结构
- **结构模板**：可将常用的文件夹结构保存为模板，一键加载
  - 模板按资源类型（角色卡/预设/世界书）分开存储，互不干扰
  - 支持保存、加载、删除模板

### 批量删除

- 进入删除模式后可多选文件夹
- **级联选择**：选中父文件夹自动选中所有子文件夹
- **框选模式**：点击起点再点击终点，范围内全部选中（也支持 Shift+点击）
- **反选**：支持在全部文件夹或指定父级范围内反选
- 删除时可选择"仅移除文件夹"或"同时删除酒馆标签"
- 子文件夹自动提升到上级

## 🌳 原生界面集成

### 文件夹树快捷入口

- 在酒馆原生的**角色卡管理界面**、**世界书界面**、**预设界面**、**美化主题界面**、**背景界面**和**User 面板**中，各增加了一个文件夹树图标按钮（QR 标签页通过资源管理器内的标签页直接访问）
- 点击即可快速打开资源管理器并跳转到对应标签页
- 无需手动切换，从原生界面一键进入文件夹管理

### 原生界面文件夹过滤

- 在酒馆原生的**角色卡列表**、**预设选择器**、**世界书选择器**、**美化主题选择器**、**背景选择器**、**User 头像列表**旁注入文件夹过滤按钮
- 点击弹出文件夹树浮动面板，选择文件夹后仅显示该文件夹（含子文件夹）内的资源
- 支持「显示全部」快速取消过滤，支持「未归类」筛选未分组的资源
- 面板内支持展开/收起全部，文件夹旁显示递归计数
- 角色卡和 User 过滤通过显示/隐藏实现，预设和世界书过滤通过 detach/append option 实现，兼容 select2 等美化插件
- 角色卡列表翻页后自动重新应用过滤状态

### 原生功能增强

- 在角色卡原生的**连接到世界书**选择器旁，也会注入文件夹过滤按钮，方便在大量世界书中快速定位
- 在原生**预设详情**界面旁注入**预设条目激活分组**入口，可直接管理当前预设的条目组合
- 正则互通或新建插入后，会同步刷新原生正则面板，减少手动刷新页面的需要

## 📤 资源导出与删除

- 在资源管理器中可进入**导出模式**（绿色主题），批量选中资源后一键导出
  - 角色卡导出为 `.png` 文件
  - 预设、世界书和美化主题导出为 `.json` 文件，背景导出为原始图片文件
  - 支持全选、框选、Shift+点击多选
- 在资源管理器中可进入**删除模式**（红色主题），批量选中资源后一键删除
  - 删除前有确认弹窗，防止误操作
  - 支持全选、框选、Shift+点击多选

---

## 📦 导入 / 导出

- **导出**：可选导出全部数据或仅导出当前标签页的数据
- **导入**：从备份 JSON 文件恢复，按名称智能匹配
  - 自动创建缺失的文件夹结构
  - 按角色 avatar 匹配角色卡分配
  - 按名称匹配预设、世界书和美化主题分配
  - 恢复收藏状态和资源备注（美化、背景、预设、世界书、User）
  - 导入完成后显示详细报告（匹配数/跳过数）
- **资源导入**：八个标签页（角色卡/世界书/预设/正则/美化/背景/User/QR）均支持直接导入资源文件
  - 导入的资源自动放入当前选中的文件夹
  - 支持批量导入多个文件
  - 重名检测：可选跳过、覆盖或自动重命名

## 📖 角色世界书归类

在世界书标签页左栏提供**角色世界书归类**按钮，一键管理角色卡关联的世界书：

- **关联世界书扫描**：自动扫描所有角色卡的 `extensions.world` 字段，列出已关联的世界书及对应角色
- **内嵌世界书提取**：扫描角色卡内嵌的 `character_book` 数据，可一键提取导入到世界书列表
- **批量归类**：勾选需要归类的世界书，选择目标文件夹，一键完成归类
- **自动提取设置**：启用后，通过资源管理器导入角色卡时，会自动提取内嵌世界书并归类到设定的文件夹
- 目标文件夹设置会持久化保存，下次打开弹窗自动恢复

## 🎛️ 世界书激活分组

在世界书标签页提供**激活分组**管理功能，可将多本世界书组合为一个分组，实现一键批量激活/关闭，并支持绑定角色卡或预设以自动切换：

### 分组管理

- **创建分组**：输入分组名称，从世界书列表中勾选需要包含的世界书，保存为一个激活分组
- **编辑分组**：修改分组名称、增删包含的世界书
- **编辑弹窗文件夹过滤**：编辑分组的世界书列表时，支持按文件夹筛选世界书，方便在大量世界书中快速定位
- **删除分组**：删除不再需要的分组

### 手动应用与取消

- **应用（▶）**：一键激活分组内所有世界书
  - 若已有其他分组处于应用状态，弹出确认对话框，可选择**叠加**（保留已有分组，追加新分组）或**替换**（关闭已有分组的独占世界书，再应用新分组）

  - 与其他已应用分组共享的世界书保持激活，避免误关

### 绑定与自动应用

- **绑定角色卡**：将分组绑定到一个或多个角色卡，切换到绑定角色时自动激活该分组
- **绑定预设**：将分组绑定到一个或多个预设，切换到绑定预设时自动激活该分组
- **作用域**：分组可设为「全局」（始终参与自动应用判断）或「仅绑定时」（仅在匹配绑定条件时生效）
- **自动激活/关闭**：切换角色或预设时，自动检测所有分组的绑定条件，激活匹配的分组、关闭不再匹配的分组
- **详细提示**：自动应用时显示详细 toast 通知，说明每个分组被激活/关闭/保持的原因（如「匹配角色「Alice」→ 开启分组「RPG设定」」）

### 重命名同步

- 角色卡重命名时，自动同步所有分组中绑定的角色卡引用
- 预设重命名时，自动同步所有分组中绑定的预设引用
- 无需手动更新绑定关系

每条聊天记录右侧提供以下操作按钮：

## 📝 技术说明

- 角色卡文件夹基于酒馆原生标签系统，文件夹 = 标签
- 预设、世界书和美化主题的分组数据存储在 `extension_settings` 中，不依赖标签
- 自定义布局数据存储在 `extension_settings` 的 `customLayout` 字段中
- 自定义外观数据按当前酒馆主题名存储在 `extension_settings` 的 `customStyles` 中；用户保存的外观预设存储在 `userPresets` 中
- 界面语言设置存储在 `extension_settings` 的 `language` 字段中
- 条目缝合完成后的跳转策略存储在 `extension_settings` 的 `entryTransferPostAction` 字段中
- 移动端顶部栏避让开关存储在 `extension_settings` 的 `mobileTopbarAvoid` 字段中
- 备注数据存储在 `extension_settings` 的 `themeNotes`、`bgNotes`、`presetNotes`、`worldInfoNotes`、`personaNotes`、`qrNotes` 字段中
- 聊天记录备注数据存储在 `extension_settings` 的 `chatNotes` 字段中，以聊天文件名（不含 `.jsonl`）为键
- 置顶聊天列表存储在 `extension_settings` 的 `pinnedChats` 数组中，每项为 `{ avatar, chatFileName }`
- User（Persona）的绑定角色/群组数据存储在 `power_user.persona_descriptions[avatarId].connections` 中
- 世界书激活分组数据存储在 `extension_settings` 的 `wiActivePresets` 数组中，每个分组包含 `{name, books, scope, bindChars, bindPresets}` 字段
- 预设条目激活分组数据存储在 `extension_settings` 的 `presetDetailActivePresets` 中；`_presetDetailAppliedPresetIndices` 记录各预设当前已应用的分组索引
- 正则脚本文件夹数据存储在 `extension_settings` 的 `regexFolderTree` 和 `regexGlobalGroups` 字段中；角色 / 预设正则仍沿用酒馆原生字段，互通时会一并同步
- 正则激活分组数据存储在 `extension_settings` 的 `regexActivePresets` 数组中，每个分组包含 `{name, scripts}` 字段；`_regexAppliedPresetIndices` 数组追踪当前已应用的分组索引
- 快速回复集文件夹数据存储在 `extension_settings` 的 `resourceFolderTree.quickreply` 和 `resourceGroups.quickreply` 字段中
- 快速回复激活分组数据存储在 `extension_settings` 的 `qrActivePresets` 数组中，每个分组包含 `{name, sets, scope, bindChars, bindPresets}` 字段
- 所有配置通过 `saveSettingsDebounced` 持久化，无需手动保存
- 自动清理角色身上多余的路径标签（只保留最深层叶子标签）
